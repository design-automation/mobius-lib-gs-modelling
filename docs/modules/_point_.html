FromPointsMean

  • FromPointsMean(points: IPoint[]): IPoint
    • Defined in point.ts:66

    Adds a point that is the center of a list of points

    Calculates the mean of the X, Y and Z coordinates of the list of points and returns a point using the results
    Point returned is the three dimensional centroid of the specified list of points
    If points inputed are co-planar, this returns their area centroid
    If two points are inputed, this returns their mid-point

    Parameters

    • points: IPoint[]

      List of points

    Returns IPoint

    New point if successful, null if unsuccessful or on error
